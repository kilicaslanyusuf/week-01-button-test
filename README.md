# Collector Rush

Küçük bir 2D arcade mini oyun.

## Nasıl oynanır?
- Oyunu başlatmak için Enter
- Hareket için yön tuşları
- Süre bitmeden mümkün olduğunca yüksek skor yap

## Oyun mantığı
- Collectible: +1 skor
- Hazard: -3 saniye
- Bonus: +2 saniye
- Bonus süresi maksimum 30 saniyeye kadar çıkar

## Oyun sonu
- Süre bittiğinde skorun, derecen ve en iyi skorun gösterilir
- Enter ile tekrar başlayabilirsin
